  • [^...] checks if any one character between the brackets is not present in the searched string.
  • [0-9] matches any decimal digit from 0 through 9.

  • [A-Z] matches any character from uppercase A through uppercase Z.
  • [a-zA-Z] matches any character from lowercase a through uppercase Z.

Page #: Assertions

  • "^" matches the beginning of input. The pattern to match must appear after the caret symbol as in "^a".

    If the multiline flag, '/m', is used, then it also applies immediately after a line break. Note that the caret symbol, '^', means negation when used at the start of a range.

  • "$" matches the end of input. The pattern to match must appear before the caret symbol as in "a$".

    If the multiline flag, "/m", is used, then it also applies immediately before a line break.

  • "\b" matches a word boundary. This is the position where a word character is not followed or preceded by another word-character.

    To be precise, \b[pattern] will match a word starting pattern whereas [pattern]\b will match a word ending pattern

  • "\B" matches a non-word boundary. This is a position where the previous and next character are of the same type: Either both must be words, or both must be non-words.

    To be precise, \B[pattern] will require that the beginning of the pattern be at a non-word boudary whereas [pattern]\B will require that the end of the pattern be at a non-word boundary. Compare this example, to this example, .

Page #: Quantifiers

  • p+ matches any string containing one or more p's. p+ is equivalent to p{1,}
  • p* matches any string containing zero or more p's. One might conclude that p* will match everything because of the "zero" reference, but that is not the case.
  • p? matches any string containing zero or one p's.
  • p{N} matches any string containing a sequence of N p's.
  • p{N1, N2} matches any string containing a sequence of N1 or N2 p's.
  • p{N,} matches any string containing at least a sequence of N p's. Note that the "," cannot be followed by any white space.
  • p$ matches any string containing p at the end of it.
  • ^p matches any string containing p at the start of it.
